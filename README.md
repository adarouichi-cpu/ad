# Habit Tracker Web App

## Description
The Habit Tracker Web App is a simple and efficient tool designed to help users track their daily habits and improve their productivity. The application allows users to manage various habits, set goals, and view progress over time.

## Technologies
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Version Control:** Git
- **Deployment:** Heroku

## Setup Instructions
1. **Clone the Repository**  
   Run the following command in your terminal to clone the repository:
   ```bash
   git clone https://github.com/adarouichi-cpu/ad.git
   cd ad
   ```  

2. **Install Dependencies**  
   Install the required dependencies using npm:
   ```bash
   npm install
   ```  

3. **Set Up the Database**  
   Create a MongoDB database and update the database connection string in the `.env` file.

4. **Run the Application**  
   Start the server using:
   ```bash
   npm start
   ```  
   The application will be running on `http://localhost:3000`.

## Usage Guidelines
- **Register an Account:**  
   Users can create an account by providing an email and password.
- **Add Habits:**  
   Once logged in, users can add new habits by selecting a title and setting a frequency.
- **Track Progress:**  
   Users can mark habits as completed and view their progress through a user-friendly dashboard.
